Quick start ​

WARNING

Please ensure that it is installedNode.js , Git and Chrome/Chromium

TIP

Can be opened in advanceQQ 开放平台register your own robot account

Download and build tools for executing instructions in the command line window

shell
npm install alemon-cli -g

Start building robots

shell
alemon-cli install

Enter the directory and load dependencies

shell
cd <robot name>
npm install

Daily operation ​

If you need to observe the console printing messages

The front desk should be kept running

shell
npm run app

If 24-hour operation is required

Should be switched to running in the background

shell
npm run start

If the environment changes

You can choose to log in again and configure the machine

shell
npm run login

Extension Instructions ​

shell
npm run stop     #stop
npm run restart  #restart
npm run delete   #delete
npm run login    #login
npm run logs     #print
npm run monit    #Monitoring management
npm run kill     #Kill instance
npm run update   #Update initiator
npm run dev      #Hot Development Launch
